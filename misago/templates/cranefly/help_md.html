Títulos

Você pode adicionar diversos tipos de título usando "hashtags" #

  # Esse é um título tipo 1; <h1>
  ## Esse é um título tipo 2; <h2>
  ...
  ###### Esse é um título tipo 6; <h6>
Resultam em: 
  
Esse é um título tipo 1; <h1>

  
Esse é um título tipo 2; <h2>

  ...
  
Esse é um título tipo 6; <h6>

Enfâses

Você pode adicionar ênfases ao seu texto usando * e _

    *Esse texto vai ser itálico*
    _Esse texto também vai ser itálico_
    **Esse texto vai ser em negrito**
    __Esse texto também!__
    *Você **pode** combinar eles*
Resultam em: 
    Esse texto vai ser itálico
    Esse texto também vai ser itálico
    Esse texto vai ser em negrito
    Esse texto também!
    Você pode combinar eles

Listas

Você pode criar listas usando * antes dos items

    * Item
    * Outro Item
Resultam em: 
  
      
  •  Item 
    • 
      
  •  Outro Item
    •

    Links

    Você pode adicionar links usando o formato [Texto](URL)

        [ApenasAlunos!](http://www.apenasalunos.com.br/)
    Resultam em: 
        ApenasAlunos!

    Imagens

    Você pode adicionar Imagens usando o formato ![Texto](URL da Imagem)

        ![Gatinho](https://cdn.pixabay.com/photo/2015/08/30/10/58/cat-914110_960_720.jpg)
    Resultam em:

    Citações

    Você pode adicionar citações com >

        > isso é uma citação
    >> isso é uma citação dentro de uma citação
    Resultam em:
    isso é uma citação
    isso é uma citação dentro de uma citação

    Códigos

    Você pode adicionar snippets de código com ` e ```

        `isso é um código inline`
    ```
    isso é um bloco de código
    ```
    Resultam em:
    isso é um código inline 
    isso é um bloco de código
